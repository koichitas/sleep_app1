# Store Listing - ગુજરાતી (gu)

---

## એપ શીર્ષક (મહત્તમ 30 અક્ષરો)

```
Doze Panels: ઊંઘ નંબર ગેમ
```

---

## સંક્ષિપ્ત વર્ણન (મહત્તમ 80 અક્ષરો)

```
મગજને થકવી નૈસર્ગિક રીતે ઊંઘ લાવવામાં મદદ કરતી નંબર પઝલ.
```

---

## સંપૂર્ણ વર્ણન (મહત્તમ 4000 અક્ષરો)

```
તમને ઊંઘ પાડી દે તેવી ગેમ — ઈરાદાપૂર્વક.

ઊંઘ નથી આવતી? મધ્યરાત્રે પણ સ્ક્રોલ કરો છો? Doze Panels સૂવાના સમયને હળવી મગજ કસરતમાં ફેરવે છે. ધ્યાન નહીં, સફેદ અવાજ નહીં — ફક્ત નંબર.

ક્રમ પ્રમાણે નંબર ટૅપ કરો. મગજ થાકે. ઊંઘ આવે.

નંબર સ્ક્રીન પર વેરવિખેર હોય છે. 1 થી ક્રમ પ્રમાણે ટૅપ કરવાનું. સાચા નંબર અદ્રશ્ય નથી થતા. આગળ શોધતી વખતે ક્યાં રોક્યા હતા તે યાદ રાખવું. આ માનસિક પ્રયત્ન ધીમે ધીમે મગજ થકવે.

તબક્કો 1 (4×4): ટૅપ કરેલા નંબર લીલા રહે.
તબક્કો 2 (5×5)+: એક સેકન્ડ પછી લીલો રંગ જાય.
છેલ્લો તબક્કો (10×10 — 100 નંબર): નાના પ્લ‌ૅ‌ન, ઝીણી આંખ.

10 મિનિટ કોઈ ક્રિયા ન હોય તો ઊંઘી ગયા ગણાય.

ઊંઘ આવવામાં તકલીફ પડતી હોય, શાંત ઊંઘની દિનચર્યા ઇચ્છતા હોય તેમના માટે આ ગેમ ઉત્તમ છે.
```
