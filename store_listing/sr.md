# Store Listing - Српски (sr)

---

## Назив апликације (макс. 30 знакова)

```
Doze Panels: Игра бројева за сан
```

---

## Кратак опис (макс. 80 знакова)

```
Бројчана загонетка која умарa ум и помаже да се природно заспи.
```

---

## Потпун опис (макс. 4000 знакова)

```
Игра која вас успављује — намерно.

Не можете да заспите? Скролујете у поноћ? Doze Panels претвара спавање у лагану вежбу за мозак. Без медитације, без беле буке — само бројеви.

Додирните бројеве по реду. Мозак се умара. Сан долази.

Бројеви су расути по екрану. Задатак: додирнути их од 1 по реду. Тачни бројеви не нестају. Приликом тражења следећег треба памтити где сте стали. Тај умни напор постепено умара мозак.

Ниво 1 (4×4): Додирнути бројеви остају зелени.
Ниво 2 (5×5)+: Зелена нестаје после секунде.
Завршни ниво (10×10 — 100 бројева): Мали панели, сужене очи.

10 минута без активности — евидентира се као заспало.

За оне са тешкоћама при спавању или оне који желе миран ноћни ритуал.
```
