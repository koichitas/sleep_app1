# Store Listing - Русский / Россия (ru-RU)

---

## Название приложения (макс. 30 символов)

```
Doze Panels: Игра для сна
```

---

## Краткое описание (макс. 80 символов)

```
Числовая головоломка, утомляющая разум и помогающая заснуть естественно.
```

---

## Полное описание (макс. 4000 символов)

```
Игра, которая усыпляет вас — намеренно.

Не можете заснуть? Продолжаете листать экран в полночь? Doze Panels превращает время отхода ко сну в лёгкую умственную тренировку, которая естественным образом приводит к засыпанию. Никакой медитации, никакого белого шума — только цифры.

Нажимайте цифры по порядку. Мозг устаёт. Сон приходит.


■ Как это работает

Цифры разбросаны по экрану. Ваша задача: нажимать их по порядку, начиная с 1.

Просто — но с изюминкой. Правильные цифры не исчезают. Каждая нажатая цифра остаётся на экране, поэтому нужно помнить, где остановились, пока ищете следующую. Это умственное усилие — искать, помнить, снова искать — постепенно изматывает мозг.

Правильное нажатие: вспыхивает зелёным. Неправильное: вспыхивает красным. Продолжайте, пока веки не отяжелеют.


■ Уровни, которые растут вместе с вами

Начинайте легко. Засыпайте по мере продвижения.

▶ Уровень 1 (4×4 — 16 цифр)
Нажатые цифры остаются зелёными, и вы всегда знаете свой прогресс. Идеально для разминки — или для ночей, когда вы уже наполовину спите.

▶ Уровень 2 (5×5) и далее
Зелёный исчезает через секунду. Теперь нужно запоминать самостоятельно. По мере продвижения количество цифр растёт, а каждая панель становится меньше.

▶ Финальный уровень (10×10 — 100 цифр)
Сто цифр. Крошечные панели. Прищуренные глаза, напряжённый мозг. Если вы дошли сюда, не заснув, попробуйте ещё раз.


■ Подсказка

Промахнулись три раза подряд? Можно продолжить самостоятельно или посмотреть короткую рекламу, чтобы узнать следующую правильную цифру.


■ Определение сна

Нет активности 10 минут? Приложение решает, что вы заснули — и поздравляет вас.


■ Для кого это?

· Те, кому трудно засыпать или кто страдает бессонницей
· Все, кто хочет спокойный ритуал отхода ко сну
· Те, кто засыпает с экраном в руках
· Любители лёгких умственных упражнений или числовых головоломок
· Все, кто хочет прекратить бесконечный скроллинг ночью

Если умеете считать — умеете играть. Все возрасты приветствуются.
```
