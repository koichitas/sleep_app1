# アプリコンセプト（翻訳共通原本）

各言語の掲載情報を作成する際は、このファイルのコンセプトとゲームルールを参照すること。
ニュアンスや意図を正確に伝えるための原本。

---

## 目的・ターゲット

- 寝つきが悪い人、なかなか眠れない人のためのアプリ
- 毎晩同じルーティンで眠りたい人
- スマホを見ながら自然に眠りたい人
- 脳トレとして楽しみたい人（サブターゲット）

## 眠れる仕組み（コアメカニズム）

- 数字を探す・覚えるという「適度な脳の使用」によって自然な疲労感を生む
- 徐々に難しくなることでイライラせず、適切な負荷が続く
- ステージクリア後の広告（= 何もしなくていい休憩時間）が眠気を後押しする
- 「脳疲労 → 休憩 → 眠気」というサイクルを繰り返す設計

---

## ゲームルール（詳細・正確版）

- 画面に並んだ数字を1から順番にタップするだけ
- 正解した数字は消えずに残る（どこまで進んだか自分で覚える必要がある）
- 正解 → 緑色に約1秒光る / 不正解 → 赤色に約1秒光る
- ステージ1（4×4）のみ：正解した数字の緑色が残り続ける（初心者向けの補助）
- 3回連続で不正解になると、モーダルが表示される。そのまま続けるか、広告を見てヒントをもらうかを選べる。ヒントを選ぶと次の正解の数字が緑色に光る
- 10分間操作なし → 寝落ち判定。お祝いのメッセージ表示後、スタート画面に戻る
  - ※メッセージは言語ごとに異なるため、翻訳時は「寝落ちを祝うメッセージ」として適宜表現すること

## ステージ構成

| ステージ | グリッド | 数字の数 | 特徴 |
|---|---|---|---|
| 1 | 4×4 | 16 | 正解した数字の色が残る（易しい） |
| 2 | 5×5 | 25 | 色は1秒で消える（記憶が必要） |
| 3 | 6×6 | 36 | 難易度上昇 |
| 4 | 7×7 | 49 | |
| 5 | 8×8 | 64 | |
| 6 | 9×9 | 81 | |
| 7 | 10×10 | 100 | 最終ステージ。パネルが小さくなり目も疲れる |

## 記録機能

スタート画面に直近のプレイ記録を表示：
- 寝落ち（何分後に寝落ちしたか）
- クリア（全ステージクリアにかかった時間）
- 途中離脱（アプリを閉じた）

## タイムアタック（サブ機能）

- クリアタイムを記録・表示（サーバー保存なし、自分との闘い）
- 眠れなかった夜はタイムアタックとして楽しめる

## トーン・ブランドイメージ

- 穏やか、ゆったり、プレッシャーなし
- 「失敗してもOK」「寝落ちすること自体が成功」というポジティブなメッセージ
- 暗い画面・ダークテーマで就寝前の使用に配慮
- 全年齢対象：数字がわかれば誰でも遊べる
